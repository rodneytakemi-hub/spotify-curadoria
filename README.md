# Spotify Curadoria

Este projeto analisa 7 anos de histórico de escuta pessoal no Spotify — 32 mil execuções, 9 mil músicas únicas. O passado é explorado através de análises e visualizações interativas de comportamento musical. O presente é interpretado por técnicas de machine learning: Item2Vec, PCA, GMM e Cadeias de Markov. O futuro são três playlists personalizadas, geradas a partir de tudo isso.

🎵 [Ver resultados do projeto](https://rodneytakemi.github.io/spotify-curadoria/)

- [NB1 — Engenharia de features](https://nbviewer.org/github/rodneytakemi/spotify-curadoria/blob/main/notebooks/nb1.ipynb): leitura dos dados brutos, unificação de nomes, sessões e features comportamentais

- [NB4 — Modelagem](https://nbviewer.org/github/rodneytakemi/spotify-curadoria/blob/main/notebooks/nb4.ipynb): Item2Vec, PCA, GMM e Cadeias de Markov para geração das playlists

# Spotify Curation

This project analyzes 7 years of personal Spotify listening history — 32,000 plays, 9,000 unique tracks. The past is explored through interactive visualizations of musical behavior. The present is interpreted through machine learning: Item2Vec, PCA, GMM, and Markov Chains. The future is three personalized playlists, generated from all of it.

🎵 [View project results](https://rodneytakemi.github.io/spotify-curadoria/)

## Notebooks

- [NB1 — Feature Engineering](https://nbviewer.org/github/rodneytakemi/spotify-curadoria/blob/main/notebooks/nb1.ipynb): raw data processing, name unification, session detection and behavioral features
- [NB2 — Spotify API](https://nbviewer.org/github/rodneytakemi/spotify-curadoria/blob/main/notebooks/nb2.ipynb): metadata collection via API — track duration and release date
- [NB3 — Execution Features](https://nbviewer.org/github/rodneytakemi/spotify-curadoria/blob/main/notebooks/nb3.ipynb): retention rate, full listen rate, skip rate and intentional start
- [NB4 — Modeling](https://nbviewer.org/github/rodneytakemi/spotify-curadoria/blob/main/notebooks/nb4.ipynb): Item2Vec, PCA, GMM and Markov Chains for playlist generation
- [NB5 — Dashboards](https://nbviewer.org/github/rodneytakemi/spotify-curadoria/blob/main/notebooks/nb5.ipynb): annual retrospectives, listening habit analysis and playlist presentation

## Stack

Python · Pandas · Gensim · scikit-learn · Plotly · Google Colab
